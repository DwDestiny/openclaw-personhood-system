# Inbox - 待验证认知候选

> 本目录存放尚未验证稳定的新认知候选。
> 不直接升级到长期文档，先在这里沉淀观察。

## 使用说明

- 新认知先写到这里，不要直接污染长期文档
- 经过多次验证后再升级到 `world.md`、`self.md` 或 `expression.md`
- 定期清理已失效或已过时的候选

## 文件命名规范

```
{date}-{type}-{简短描述}.md

示例：
2026-03-17-trend-AI-agent.md
2026-03-17-self-proactive-remind.md
2026-03-17-expression-emoji-limit.md
```

## 类型标签

- `trend`: 行业/平台趋势判断
- `self`: 自我认知新发现
- `expression`: 表达风格新尝试
- `skill`: skill 化候选
- `workflow`: workflow 化候选
